Filesystem commands:

* `pwd`. Imprime en el terminal el directorio actual
* `~`. = `$HOME` Indica la carpeta home del usuario actual
* `ls`. Lista los contenidos del directorio actual o de otro si se le indica, ejemplo `ls /lib`.
